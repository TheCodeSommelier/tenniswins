## SCSS Conventions

```c-dashboard``` == Container Dashboard

```__element``` == Element

```--modifier``` == Modifier

```
  .c-button {

    /* Element -- will result in a class c-button__anchor */
    &__anchor {
      /* styles here */
    }

    /* Modifier -- will result in a class c-button--outline */
    &--outline {
      /* styles here */
    }
  }
```
